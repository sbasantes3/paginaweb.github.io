Cloud Computing

Conceptos

¿Qué es Cloud Computing? Servicio que permite acceder a recursos como servidores, bases de datos, redes y software a través de Internet.

SaaS: Software como servicio.

PaaS: Plataforma como servicio.

IaaS: Infraestructura como servicio.

¿Qué es Virtualización? Proceso que permite ejecutar varias máquinas virtuales sobre un mismo hardware físico.

¿Qué es Docker? Plataforma de contenedores que permite crear, ejecutar y desplegar aplicaciones aisladas.

Anydesk y TeamViewer

Anydesk: Herramienta para acceso remoto a computadoras.

TeamViewer: Software de acceso remoto y soporte técnico.
